Working Security

Security guidelines for flexible working environments

Remote & Client Work

Security practices for working from home or at client sites

Working Remotely

Physical Security

  • Work in a private space where screens are not visible to others
  • Lock your computer when stepping away
  • Store company devices securely when not in use
  • Use privacy screens when required
  • Shred confidential documents

Network Security

  • Use a secure home Wi-Fi network with WPA2 or WPA3 encryption
  • Change default router passwords and keep firmware updated
  • Use company VPN when accessing internal systems
  • Separate work and personal network traffic where possible
  • Avoid conducting sensitive work on shared family devices

Communication Security

  • Be mindful of who can overhear calls and meetings
  • Use headphones for confidential discussions
  • Blur or use virtual backgrounds if needed
  • Close confidential documents before screen sharing
  • Mute when not speaking to prevent background recording

Client Sites

Key Principle

When working at a customer site, follow both company security policies and the customer's security requirements. If there is a conflict, follow the stricter policy or consult your manager.

Before Visiting a Client Site

  1. Review the customer's security policies and access requirements
  2. Ensure your device meets customer security standards
  3. Confirm you have necessary credentials and approvals
  4. Understand customer data handling rules and export restrictions
  5. Know who to contact for security issues at the client site

While at the Client Site

  • Always wear visible identification badges as required
  • Do not allow others to tailgate through secure doors
  • Follow customer policies for device usage and network access
  • Do not photograph or document facilities without permission
  • Be aware of clean desk policies and secure your work area
  • Use customer-provided equipment or networks when required
  • Report any security incidents to both customer and company contacts

Data Handling at Client Sites

  • Do not remove customer data from the site without explicit authorization
  • Follow customer requirements for data storage and transfer
  • Use only approved tools and methods for accessing customer systems
  • Respect data export controls and cross-border restrictions
  • Delete local copies of customer data when work is complete

Device Security

Keeping your devices secure and up-to-date

Essential Security Measures

Operating System Updates

Install security updates within 48 hours of release. Enable automatic updates where possible.

Antivirus & Anti-Malware

Keep protection software active and updated. Run regular scans.

Disk Encryption

Full disk encryption must be enabled on all devices containing company data.

Screen Lock

Set automatic screen lock after 5 minutes of inactivity. Use strong PIN or biometrics.

Firewall Protection

Keep device firewalls enabled at all times. Do not create exceptions without IT approval.

Device Hygiene Practices

Software Management

  • Only install approved business software
  • Keep all applications updated
  • Uninstall unused applications
  • Avoid pirated or cracked software

Physical Care

  • Never leave devices unattended in public
  • Use cable locks in shared spaces
  • Report lost or stolen devices immediately
  • Keep devices in secure bags when traveling

Data Backup

  • Back up important data regularly
  • Use company-approved backup solutions
  • Test backup restoration periodically
  • Store backups securely and encrypted

Mobile Devices

  • Enable remote wipe capabilities
  • Use biometric authentication
  • Be cautious with app permissions
  • Avoid jailbreaking or rooting

Bring Your Own Device (BYOD)

  1. Personal devices used for work must be enrolled in company device management
  2. Apply the same security standards as company devices
  3. Separate work and personal data when possible
  4. Understand that IT may remotely wipe company data if the device is lost
  5. Contact IT before using a personal device for work

Network Safety

Staying safe on public Wi-Fi and unsecured networks

Public Wi-Fi Risks

Risks Include

  • Man-in-the-middle attacks
  • Packet sniffing and eavesdropping
  • Evil twin networks
  • Malware distribution
  • Session hijacking

Protection Measures

  • Always use company VPN
  • Verify network authenticity
  • Use HTTPS for all websites
  • Disable auto-connect features
  • Turn off file sharing

Secure Network Practices

  1. VPN is Mandatory: Connect to company VPN before accessing resources
  2. Verify Network Name: Confirm with staff to avoid fake hotspots
  3. Forget Network: Do not save public Wi-Fi networks on your device
  4. Avoid Sensitive Tasks: Do not access banking or enter passwords without VPN
  5. Keep Software Updated: Ensure your OS and applications are patched

Safer Alternatives

Mobile Hotspot

Use your phone's cellular connection as a personal hotspot

Cellular Data

Use 4G/5G data instead of public Wi-Fi when available

Wired Connection

Use Ethernet at hotels or offices when possible

VPN (Virtual Private Network) Usage

When VPN is Required

  • Working on public Wi-Fi
  • Accessing internal systems
  • Handling sensitive data
  • Connecting to customer networks
  • Working from untrusted locations
