Employee Resources

General rules and daily security practices for all employees

Acceptable Use

Rules for using company systems and devices

Permitted Activities

  • Using company devices and systems for work-related activities
  • Accessing approved cloud services and applications
  • Reasonable personal use during breaks (email, news, etc.)
  • Installing approved software with IT department permission

Prohibited Activities

  • Accessing, storing, or distributing illegal or inappropriate content
  • Sharing login credentials or allowing unauthorized access
  • Installing unauthorized software or applications
  • Bypassing security controls or attempting to gain unauthorized access
  • Using company resources for personal business or commercial activities

Password & Access Management

Strong passwords, MFA, and access control basics

Password Requirements
  • Use at least 12 characters
  • Include upper and lower case letters, numbers, and symbols
  • Avoid names, birthdays, and common words
  • Do not reuse passwords across work accounts
Multi-Factor Authentication (MFA)
  • Enable MFA on all business accounts
  • Prefer authenticator apps over SMS when possible
  • Never approve an MFA request you did not initiate
Access Control Principles
  • Use least-privilege access
  • Request access only when needed for work
  • Report suspicious login behavior immediately

Email Security

Phishing awareness and safe email practices

Recognizing Phishing Attempts

Red Flags

  • Unexpected urgent requests
  • Suspicious sender addresses
  • Spelling and grammar errors
  • Requests for credentials or payment
  • Unfamiliar links or attachments

What to Do

  • Verify sender through other channels
  • Hover over links to check URLs
  • Do not open suspicious attachments
  • Report suspicious emails to IT
  • When in doubt, ask IT first

Safe Email Practices

  • Use BCC when sending to large distribution lists to protect privacy
  • Never send passwords or sensitive data via unencrypted email
  • Double-check recipients before sending sensitive information
  • Be cautious with auto-forwarding rules and delegated access
  • Use encryption tools for confidential client communications

Cloud Usage & Document Sharing

Guidelines for secure use of cloud tools and services

Approved Cloud Services

✔ Microsoft 365 / OneDrive
✔ Google Workspace
✔ Company SharePoint
✔ Approved project management tools

Secure Document Sharing

  1. Set appropriate permissions: Only share with specific people who need access
  2. Use expiration dates: Set links to expire when no longer needed
  3. Avoid public links: Never use “anyone with the link” for sensitive documents
  4. Review regularly: Audit shared documents and revoke unnecessary access
  5. Use passwords: Add password protection for highly sensitive files
Important: Never store customer data in personal cloud accounts. Always use company-approved services.
